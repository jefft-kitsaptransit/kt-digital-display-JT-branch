Project Layout: 

App.py: the main python script that grabs KT gtfs real time feed. Also process static gtfs files as fall back. 

->templates->index.html: Front end that displays all the data. 

->static->: contains Kitsap Transit gtfs files.

Current Limitations:
Program is currently written to read static gtfs files from the directory. If we make a change to our static gtfs file (time changes etc) it will currently not be reflected unless we manually updated the gtfs files on whichever server this dashboard is running from. We could improve it by checking the online location of our static gtfs feed daily at midnight and if files have been updated automatically update the copies on the server.
